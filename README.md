# Dining Rewards

> This application is based on
> the [Spring Framework Essentials](https://spring.academy/courses/spring-framework-essentials) course by Spring
> Academy. It is a sample application that demonstrates the use of Spring Framework features such as _Java
> Configuration_, _Component Scanning_, _AOP_, and more.

## Prerequisites

- JDK 11, can be installed using [sdkman](https://sdkman.io/)
- Maven, or use the provided `mvnw` wrapper
- An IDE, preferably IntelliJ IDEA

## Running the application

This is a multi-module project, so you need to run the following commands from the root directory of the project:

```shell
./mvnw clean install  # install dependencies
./mvnw test           # run tests
```
